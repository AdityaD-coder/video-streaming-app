# StreAnime - Mini Anime Streamer (MERN)

A full-stack video streaming application built using the MERN stack. 
This project focuses on understanding real-world backend architecture, API handling, and performance optimization.

---

## Status: Work in Progress

This project is currently being actively developed and explored.

### Important Note
This project started from an AI-generated skeleton. 
The current goal is to **deeply understand, refactor, and improve the codebase** by implementing core backend concepts manually.

---

## Current Progress

- Basic MERN project structure
- Frontend setup for video display
- Initial backend setup (Node.js + Express)
- Placeholder / basic API integration
- Authentication flow (JWT - partial understanding & implementation)

---

## In Progress

- Moving API calls from frontend → backend
- Understanding and implementing proper JWT authentication flow
- Debugging and refactoring AI-generated code
- Improving backend structure and routing

---

## Planned Features

- Video upload & streaming functionality
- User authentication & authorization (JWT)
- Backend API for video data handling
- Caching for improved performance
- Video recommendations (future scope)
- Scalable and modular backend architecture

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)

---

## Learning Objectives

This project is focused on:

- Understanding how full-stack applications actually work
- Learning backend API design and data flow
- Implementing JWT authentication securely
- Debugging and improving existing code
- Exploring caching strategies for performance

---

## Project Approach

Instead of building everything from scratch, this project follows a different approach:

- Start with a generated structure 
- Break it down and understand each part 
- Rebuild critical components with proper logic 
- Improve code quality and architecture
